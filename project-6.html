JS Piano

Project Brief

I am currently learning to play piano in a desperate attempt to learn an instrument and appear more interesting. I decided why not try and combine two passions and built this simple piano that can be played using the keys on a computer keyboard.

What I built

Using a mix of plain Javascript (es6) and jquery I created a small piano that is played by using keyboard inputs. It hasn't helped me with the real piano in anyway but it was a brilliant exercise in CSS styling and event handling.
