Periodic Table

Project Brief

I was in need of a periodic table to settle a friendly bet and came across a plethora of choices online. There are literally countless amounts of periodic tables to choose from. I decided one more couldn't hurt.

What I built

Using Jquery I built a clean looking, modern periodic table. Instead of utilising an API, I collated all of the data personally.
